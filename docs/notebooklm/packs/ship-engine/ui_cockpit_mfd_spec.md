Cockpit MFD — the design target for all TRENCH UI surfaces.

- Phosphor-green monochrome CRT behind physical bezel
- OSB (option select buttons) as hard-edged rectangles framing the viewport — press states are brightness shifts, not animations
- Scan lines visible
- Data rendered as vector strokes on black, not filled surfaces
- Filter response = tactical scope trace
- Morph XY = radar sweep
- Parameters read like altitude readouts — fixed-width numerals, no labels, position is meaning
- Material: matte black aluminum, phosphor burn, CRT curvature distortion at edges
- Feels bolted into a panel, backlit at 3am
