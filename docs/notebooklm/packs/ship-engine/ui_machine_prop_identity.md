The plugin UI is not an interface. It is a machine prop.

Reference world: Garry's Mod DarkRP, Source-engine utility props, reactor rooms, meth lab benches, CCTV terminals, RTS sidebars, cockpit MFDs, old FPS status bars, inventory bay logic, black-site operator hardware.

The user is not "editing parameters." They are operating a suspicious machine.

Interaction vocabulary: load, arm, cut, vent, route, stabilize, overload, accept, reject.

The screen should feel like a thing in a room that players would gather around and stare at because something dangerous or valuable is happening inside it.

Every interaction must look watchable.
